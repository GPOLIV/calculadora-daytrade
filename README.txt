COMANDOS PARA RODAR A CALCULADORA (LOCAL):

1. Abra o terminal (CMD) no Windows ou o terminal do VSCode.
2. Vá até a pasta do projeto (exemplo):
   cd C:\Users\SEU_USUARIO\Documents\CalculadoraDayTrade

3. Ative o ambiente virtual (opcional mas recomendado):
python -m venv venv
   python -m venv venv
   venv\Scripts\activate

4. Instale os pacotes necessários:
   pip install -r requirements.txt

5. Inicie a calculadora:
   streamlit run app.py
